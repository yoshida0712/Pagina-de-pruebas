Etiquetas

Las etiquetas sirven para dividir en partes una pagina, facilitando su organización y estructura.

Encabezados

Dependiendo del tamaño del texto se utilizan diferentes etiquetas de encabezados. h1 para el mas grande, hasta h6 para el mas pequeño.

Ejemplo: <h2>

Parrafos

Los parrafos se definen con la etiqueta p. Se utilizan para colocar texto.

Ejemplo: <p>

Enlaces

Los enlaces se definen con la etiqueta a. Se utilizan para crear hipervínculos a otras páginas o recursos.

Ejemplo: <a href="URL"> Nota: el href="" es necesario para colocar la ruta del enlace.

Imágenes

Las imágenes se definen con la etiqueta img. Se utilizan para mostrar imágenes en la página.

Ejemplo: <img src="ruta_de_la_imagen" alt="Descripción de la imagen">

Atributos

Los atributos se colocan dentro de las etiquetas, y le dan diferentes propiedades a las etiquetas.

href

href sirve para poder colocar un enlace a una pagina, ya sea dentro de la misma pagina donde estemos o a otra. Esta se coloca en alguna etiquetas en espeficico, ya que no sirve en todas las etiquetas.

Ejemplo: <a href="URL">

src

Si la etiqueta img sirve para especificar que se colocara una imagen, el atributo src sirve para espeficicar la ruta de la imagen para que el navegador accedar a esta
