Lista de etiquetas HTML

Nota: Para completar la informacion sobre las etiquetas, se recomienda leer sobre los atributos mencionados en esta pagina, por favor revisar el siguiente enlace: Atributos HTML
Nota 2: En html se tiene en cuenta si las etiquetas son en bloque o en linea, esto se refiere a que cuando se utiliza una etiqueta en bloque, esta ocupa todo el ancho disponible (de izquierda a derecha) y comienza en una nueva línea, mientras que una etiqueta en línea solo ocupa el espacio necesario (solo ocupa el espacio maximo que ocupe el contenido en la etiqueta) y no comienza en una nueva línea.

Encabezados

Los encabezados son la etiqueta utilizada para poder mostrar titulos y subtítulos en una página web.

Existen seis tipos de encabezados, que van desde el más grande al más pequeño, siendo <h1> el más grande y <h6> el más pequeño.

Parrafos

Estos sirven para poder visualizar textos en una página web. Como el que estas leyendo ahora mismo.

Los párrafos se definen con la etiqueta <p> y pueden contener texto, imágenes y otros elementos HTML.

Saltos de linea

Linea horizontal

Esta etiqueta funciona como un salto de línea, pero en lugar de crear un espacio vertical, crea una línea horizontal que puede ser utilizada para separar secciones de contenido.

ejemplo: <hr>

Salto de linea

Esta etiqueta se utiliza para crear un salto de línea en el texto. A menudo se usa para separar visualmente párrafos o secciones de contenido.

ejemplo: <br>

Etiqueta pre

Si se necesita hacer un salto de linea, para por ejemplo separar un mismo texto dentro de un parrafo, se utliza la etiqueta <pre>.

ejemplo: 

                Este es un texto con salto de línea
                Y tendra un formato de texto distinto

Formatos de texto

Negrita

Existen dos etiquetas que se utilizan para poner un texto en negrita, la etiqueta <strong> y la etiqueta <b>.

Estas etiquetas se usan dentro de la etiqueta <p> o cualquier otra etiqueta de texto. Y se usan dependiendo del contexto e importancia del texto.

Ejemplo:Texto en negrilla

Cursiva

Al igual que la negrita, existen dos etiquetas que se utilizan para poner un texto en cursiva, la etiqueta <em> y la etiqueta <i>.

Tambien se utilizan dentro de una etiqueta de texto, y el uso de depende del contexto, <em> se usa para dar enfasis semantico en la lectura, mientras que <i> se usa para estilos visuales o para o para casos específicos como nombres de obras, especies, términos técnicos, palabras en otro idioma, etc.

Ejemplo: Este es un texto subscript con la etiqueta sub y Este es un texto superscript con la etiqueta sup

Comentariado

Los comentarios sirven para poder dejar notas o recordatorios dentro del código HTML, que no serán visibles en la página web. Se utilizan para explicar el código o para dejar información adicional para otros desarrolladores. Tambien se pueden utilizar para desactivar temporalmente partes del código sin eliminarlas.

Ejemplo: <!-- Este es un comentario -->

Etiquetas de enlace

Etiqueta a

Esta etiqueta se utiliza para definir un hipervínculo. El atributo href se utiliza para especificar la URL del enlace. <a>

Ejemplo: Este es un enlace a youtube.

Nota: dependiendo de si la ruta de acceso a través del link, está dentro del mismo proyecto/página, o si está en otra página web, se utiliza una ruta relativa o absoluta. Esto cambia el valor del atributo href. Si está en otra página se coloca la URL de la página, si está dentro del mismo proyecto/página, se coloca la ruta con usos de ../ para subir un nivel en la jerarquía de carpetas.

Nota 4: Con el atributo title en un enlace, se puede proporcionar información adicional que se mostrará como un tooltip cuando el usuario pase el cursor sobre el enlace. <a href="https://www.ejemplo.com" title="Ir a Ejemplo">Ejemplo</a>

Nota 5: Si la pagina en la que se esta trabajando resulta muy larga, se puede utilizar el atributo id para crear marcadores dentro de la misma página. Esto permite que los enlaces se desplacen a una sección específica de la página. Por ejemplo: Ir a Sección 1 y luego en la sección correspondiente:

Botones con enlaces

Los botones también pueden funcionar como enlaces. Para hacer esto, simplemente se puede utilizar la propiedad onclick en el botón para redirigir a la URL deseada. Asi funciona el boton de volver de la pagina.

Ejemplo: <button onclick="location.href='https://www.ejemplo.com'">Ir a Ejemplo</button>

Imagenes

Img

La etiqueta <img> se utiliza para insertar imágenes en una página web. El atributo src se utiliza para especificar la URL de la imagen, y el atributo alt se utiliza para proporcionar un texto alternativo que se mostrará si la imagen no se puede cargar.

Ejemplo: <img src="imagen.jpg" alt="Descripción de la imagen">

Etiqueta Map

La etiqueta <map> se utiliza para definir un mapa de imagen, que es una imagen con áreas seleccionables. Se utiliza junto con la etiqueta <area> para definir las áreas específicas dentro de la imagen que son seleccionables.

Nota: se debe utilizar el atributo usemap en la etiqueta <img> para asociar la imagen con el mapa de imagen definido por la etiqueta <map>.

Junto a esta etiqueta se debe usar <area> para definir las áreas seleccionables dentro de la imagen, dentro de la etiqueta <map>, a la cual se hace referencia con el atributo name.

Nota: Junto a la etiqueta <area>, se usan los atributos shape y coords. El atributo shape define la forma del área seleccionable (rectángulo, círculo, polígono), y el atributo coords define las coordenadas de la forma en la imagen.

Ejemplo: <img usemap="#miMapa"> <map name="miMapa"> <area shape="rect" coords="34,44,270,350" href="imagen.jpg"> </map>

Picture

La etiqueta <picture> se utiliza para proporcionar imágenes responsivas en HTML. Permite definir múltiples fuentes de imagen para diferentes condiciones, como el tamaño de la pantalla o la resolución del dispositivo. Se utiliza con la etiqueta <source> para especificar las diferentes fuentes de imagen y la etiqueta <img> como una fuente de respaldo.

ejemplo: <picture> <source media="(min-width: 650px)" srcset="imagen-grande.jpg"> <source media="(min-width: 465px)" srcset="imagen-mediana.jpg"> <img src="imagen-pequena.jpg" alt="Descripción de la imagen"> </picture>

Source

La etiqueta <source> se utiliza dentro de la etiqueta <picture> para definir diferentes fuentes de imagen para diferentes condiciones. Permite especificar diferentes archivos de imagen y sus condiciones de visualización, como el tamaño de la pantalla, gracias a que se permite colocar múltiples fuentes de imagen con diferentes condiciones.

Ejemplo: <source media="(min-width: 650px)" srcset="imagen-grande.jpg">

Input

La etiqueta <input> se utiliza para crear campos interactivos en un formulario HTML. Permite a los usuarios ingresar datos, como texto, selecciones o archivos.

Etiquetas usadas en la creacion de tablas

Las etiquetas a continuacion son utilzadas en conjunto para crear tablas en html

table

La etiqueta <table> se utiliza para crear una tabla en HTML. Define la estructura de la tabla y debe contener al menos una fila de encabezado o de datos.

tr

La etiqueta <tr> se utiliza para definir una fila en una tabla HTML.

th

La etiqueta <th> se utiliza para definir una celda de encabezado en una tabla HTML. El contenido de una celda de encabezado generalmente se muestra en negrita y centrado.

td

La etiqueta <td> se utiliza para definir una celda de datos en una tabla HTML. El contenido de una celda de datos se muestra normalmente en texto normal.

Forma de uso

Las etiquetas <table>, <tr>, <th> y <td> se utilizan en conjunto para crear tablas en HTML. La etiqueta <table> define la tabla, <tr> define las filas, <th> define las celdas de encabezado y <td> define las celdas de datos.

ejemplo: 

                
                <table> 
                    <tr> 
                        <th>Encabezado 1</th>
                        <th>Encabezado 2</th> 
                    </tr> 
                    <tr> 
                        <td>Dato 1</td>
                        <td>Dato 2</td> 
                    </tr> 
                </table>

colgroup

Esta etiqueta se utiliza para agrupar una o más columnas en una tabla HTML. Permite aplicar estilos o atributos a un grupo de columnas en lugar de hacerlo individualmente a cada celda. Se utiliza junto con la etiqueta <col> para definir las columnas dentro del grupo.

Ejemplo: <colgroup><col span="2" style="background-color: #f00;"></col></colgroup>

Nota: El atributo span se utiliza para especificar el número de columnas que abarca el elemento <col>.

col

Esta etiqueta se utiliza para definir una columna en una tabla HTML. Permite aplicar estilos o atributos a una columna específica en lugar de hacerlo individualmente a cada celda. Se utiliza dentro de la etiqueta <colgroup> para definir las columnas dentro del grupo.Ejemplo arriba.

Listas

Lista ordenada

Estas lista se compone de 2 etiquetas, la etiqueta <ol> que define la lista ordenada, y la etiqueta <li> que define cada elemento de la lista.

Ejemplo: <ol><li>Elemento 1</li><li>Elemento 2</li></ol>

Lista desordenada

Estas lista se compone de 2 etiquetas, la etiqueta <ul> que define la lista desordenada, y la etiqueta <li> que define cada elemento de la lista.

Ejemplo: <ul><li>Elemento 1</li><li>Elemento 2</li></ul>

Nota: La diferencia entre una lista ordenada y una desordenada es que la lista ordenada utiliza números o letras para enumerar los elementos, mientras que la lista desordenada utiliza viñetas o puntos.

Nota 2: En las listas desordenadas y ordenadas, se pueden escoger mediante estilos de css que tipo de viñetas, puntos, letra o numero se utilizan, y ademas se pueden realizar listas anidadas, es decir, listas dentro de listas.

Nota 3: Para las listas ordenadas, se puede escoger mediane el atributo start en que numero empezar la lista.

Lista descriptiva

Esta lista se compone de 3 etiquetas, la etiqueta <dl> que define la lista descriptiva, la etiqueta <dt> que define cada término de la lista, y la etiqueta <dd> que define la descripción de cada término.

Ejemplo: <dl><dt>Término 1</dt><dd>Descripción del término 1</dd></dl>

iframe

La etiqueta <iframe> se utiliza para mostrar contenido de otras fuentes dentro de la página actual. Permite mostrar contenido de otras fuentes, como videos, mapas o páginas web completas, en un área específica de la página.

Ejemplo: <iframe src="https://www.ejemplo.com"></iframe>

Nota: se puede utilizar esta etiqueta para mostrar paginas despues de ser seleccionadas por el usuario, como por ejemplo, al hacer click en un enlace que redirige a otra pagina, se puede mostrar esa pagina dentro de un iframe en lugar de abrirla en una nueva pestaña.

Ejemplo: <iframe src="demo_iframe.htm" name="iframe_a" title="Iframe Ejemplo"></iframe>

<p><a href="https://www.w3schools.com" target="iframe_a">Visitar W3Schools</a></p>

Etiquetas script y noscript

Estas etiquetas esta relacionadas con el uso del JavaScript en HTML. Mientras que la etiqueta <script> se utiliza para incluir código JavaScript en la página, la etiqueta <noscript> se utiliza para mostrar contenido alternativo cuando JavaScript está deshabilitado en el navegador del usuario.

Ejemplo: <script>alert("Hola Mundo")</script> <noscript>JavaScript está deshabilitado</noscript>

Etiquetas base

head

La etiqueta <head> se utiliza para contener información sobre el documento, como el título, la descripción y los enlaces a hojas de estilo y scripts.

Ejemplo: <head><title>Título de la página</title></head>

Nota: Hasta que se me mencione lo contrario, todas las etiquetas a continuacion van dentro de de la etiqueta <head>.

title

Esta etiqueta se utiliza para definir el título del documento. El contenido de esta etiqueta se muestra en la barra de título del navegador y en los resultados de búsqueda.

Ejemplo: <title>Título de la página</title>

style

Esta etiqueta se utiliza para definir estilos CSS dentro del documento HTML. Permite aplicar estilos a elementos específicos o a toda la página.

link

Esta etiqueta se utiliza para definir relaciones entre el documento actual y recursos externos. Se utiliza comúnmente para vincular hojas de estilo (CSS) y otros recursos.

Ejemplo: <link rel="stylesheet" type="text/css" href="estilos.css">

Nota: Esta etiqueta se utiliza tambien para colocar el favicon, que es el icono que aparece en la pestaña del navegador. Para esto se utiliza el atributo rel="icon" y el atributo href para especificar la ruta del icono.

Ejemplo de uso del favicon: <link rel="icon" type="image/x-icon" href="favicon.ico">

meta

La etiqueta <meta> se utiliza para proporcionar metadatos sobre el documento HTML. Los metadatos son datos que describen otros datos y no se muestran directamente en la página.

Nota: aunque no se menciona, la etiqueta <script> tambien se utiliza dentro de la etiqueta <head> para incluir scripts JavaScript en la página. Mas info de esta etiqueta mas arriba.
