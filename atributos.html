Lista de atributos HTML

Nota: Para complementar la informacion sobre los atributos con las etiquetas donde se pueden usar estos atributos mencionados, por favor revisar el siguiente enlace: Etiquetas HTML

Style

El atributo <style> sirve para poder colocar estilos de CSS directamente en un elemento HTML.

Por ejemplo, para poder cambiar el color de un texto, se utilizaria de la siguiente manera: <p style="color: red;">

Title

El atributo title se utiliza para proporcionar información adicional sobre un elemento HTML. Este texto aparece como un tooltip cuando el usuario pasa el cursor sobre el elemento.

Ejemplo: <abbr title="Mi Página Web">Mi Página Web</abbr>

href

El atributo <href> se utiliza en enlaces para especificar la URL a la que se dirige el enlace. Es esencial para crear hipervínculos en HTML.

Ejemplo: <a href="https://www.ejemplo.com">Enlace a Ejemplo</a>

target

El atributo <target> se utiliza en enlaces para especificar dónde se abrirá el enlace.

Este atributo puede tomar varios valores; target="_blank" abrirá el enlace en una nueva pestaña o ventana del navegador; target="_self" abrirá el enlace en la misma pestaña o ventana; target="_parent" abrirá el enlace en el marco padre, y target="_top" abrirá el enlace en toda la ventana del navegador.

Src

El atributo src se utiliza para especificar la URL de un recurso externo, como una imagen o un script. Es comúnmente utilizado en etiquetas como <img> y <script>.

Ejemplo: <img src="imagen.jpg">

alt

El atributo alt se utiliza en imágenes para proporcionar un texto alternativo que describe la imagen. Este texto es importante para la accesibilidad y se muestra si la imagen no puede cargarse.

Ejemplo: <img src="imagen.jpg" alt="Descripción de la imagen">

usemap

El atributo usemap se utiliza en imágenes para asociar la imagen con un mapa de imagen definido por la etiqueta <map>. Que se utiliza para crear áreas seleccionables dentro de una imagen. Este atributo siempre debe iniciar con el símbolo de número (#) seguido del id del mapa.

Ejemplo: <img src="imagen.jpg" usemap="#miMapa">

name

El atributo name se utiliza para asignar un nombre a un elemento HTML. Este nombre puede ser utilizado para identificar el elemento en scripts o estilos.

Ejemplo: <input type="text" name="usuario">

shape

El atributo shape se utiliza en mapas de imagen para definir la forma de un área clickable. Puede tomar valores como rect, circle o poly. Se utiliza dentro de la etiqueta <area> para definir áreas específicas dentro de una imagen.

Ejemplo: <area shape="rect" coords="34,44,270,350" href="imagen.jpg">

coords

El atributo coords se utiliza en mapas de imagen para definir las coordenadas de un área clickable. Dependiendo del valor del atributo shape, las coordenadas se especifican de diferentes maneras.

Ejemplo: <area shape="rect" coords="34,44,270,350" href="imagen.jpg">

Nota con respecto a coords y shape

Es importante que los valores de coords sean coherentes con el valor de shape para que el área clickable funcione correctamente.

El atributo shape define la forma del área clickable, mientras que coords especifica las coordenadas de esa forma dentro de la imagen. shape puedes tener diferentes valores como rect (rectángulo), circle (círculo) o poly (polígono). Las coordenadas se definen de manera diferente según el valor de shape.

coords para un rectángulo se define como x1,y1,x2,y2, donde (x1,y1) son las coordenadas de la esquina superior izquierda y (x2,y2) son las coordenadas de la esquina inferior derecha. Para un círculo, se define como x,y,r, donde (x,y) son las coordenadas del centro y r es el radio. Para un polígono, se define como una lista de coordenadas x1,y1,x2,y2,... que representan los vértices del polígono.

Mas informacion sobre este tema: w3schools

srcset

El atributo srcset se utiliza en la etiqueta <source> para proporcionar múltiples fuentes de imagen para diferentes condiciones, como el tamaño de la pantalla o la resolución del dispositivo. Permite especificar diferentes archivos de imagen y sus condiciones de visualización, ya que el navegador selecciona automáticamente la fuente más adecuada según las condiciones actuales.

Ejemplo: <source media="(min-width: 650px)" srcset="imagen-grande.jpg">

media

El atributo media se utiliza en la etiqueta <source> para especificar condiciones de visualización para la fuente de imagen. Permite definir reglas CSS que deben cumplirse para que se utilice la fuente de imagen correspondiente.

Ejemplo: <source media="(min-width: 650px)" srcset="imagen-grande.jpg">

Atributo type

El atributo type se utiliza en la etiqueta <source> para especificar el tipo de contenido. Esto ayuda al navegador a determinar cómo manejar diferente tipos de datos, como textos, imagenes, archivos de audio, etc.

Ejemplo: <input type="image/jpeg">

colspan y rowspan

El atributo colspan se utiliza en las celdas de una tabla para especificar cuántas columnas debe abarcar. Mientras que rowspan se utiliza para especificar cuántas filas debe abarcar una celda. Estos atributos son útiles para crear tablas más complejas y organizadas.

Ejemplo: <td colspan="2">Contenido</td>. En este ejemplo, la celda abarca dos columnas.

Ejemplo: <td rowspan="2">Contenido</td>. En este ejemplo, la celda abarca dos filas.

span

El atribuno span se utiliza en la etiqueta <col> dentro de un grupo de columnas definido por la etiqueta <colgroup>. Este atributo especifica cuántas columnas debe abarcar una columna específica dentro del grupo. Es útil para definir estilos o propiedades que se aplican a múltiples columnas en una tabla.

start

Este atributo se utiliza en la etiqueta <ol> para especificar el número con el que debe comenzar la lista ordenada. Por defecto, las listas ordenadas comienzan en 1, pero con el atributo start, puedes definir un número diferente para iniciar la numeración.

Ejemplo: <ol start="5"><li>Elemento 5</li><li>Elemento 6</li></ol>

class

El atributo class se utiliza en casi todas las etiquetas HTML para especificar una o más clases a las que pertenece un elemento. Las clases se utilizan principalmente para aplicar estilos CSS a grupos de elementos o para seleccionar elementos en JavaScript. Para llamarlo desde el CSS se debe usar un punto (.) seguido del nombre de la clase.

Ejemplo: <div class="mi-clase">Contenido</div>

id

El atributo id se utiliza para asignar un identificador único a un elemento HTML. Este identificador puede ser utilizado para aplicar estilos CSS específicos o para manipular el elemento mediante JavaScript. Este es diferente a class, que puede ser compartido por múltiples elementos, ya que solo se puede asignar un único id a cada elemento en una página HTML. Para llamarlo desde el CSS se debe usar el símbolo de número (#) seguido del id del elemento.

Ejemplo: <div id="mi-id">Contenido</div>
